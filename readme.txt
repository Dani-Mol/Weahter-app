Esta aplicacion esta hecha en el lenguaje de Programacion Go

La version con la que se ejecuta este programa es la version go1.23.0

El programa fue corrido en un sistema operativo macos 15.0 (24A335)

La paqueterias que se utilizan en el programa son:

fyne.io/fyne/v2

Para instalar go se necesita descargarlo de su pagina oficial segun su 
sistema operativo
https://go.dev/dl/

Linux (Debian/Ubuntu )

Posteriormente descomprimir el archivo

Extrae el archivo tar en /usr/local:
sudo tar -C /usr/local -xzf go1.21.1.linux-amd64.tar.gz

Agrega Go a tu PATH (edita ~/.bashrc o ~/.profile):
export PATH=$PATH:/usr/local/go/bin

Actualiza tu shell:
source ~/.bashrc

Verifica la version e instalcion con:
go version

Window

Descomprimir el archivo, seguir los pasos de instalacion y 
verificar la version con 
go version

Para obtener la version de fyne utilizar el comando:
go get fyne.io/fyne/v2 en la carpeta de este proyecto
